2. Art der verarbeiteten Daten

Diese App verarbeitet folgende Kategorien personenbezogener Daten:

  • Verbindungsdaten: SSH-Server-Adressen, Ports, Benutzernamen
  • Authentifizierungsdaten: Passwörter und/oder Pfade zu SSH-Schlüsseldateien
  • Nutzungsdaten: Verbindungsverlauf und gespeicherte Profile
  • Terminaldaten: Befehlshistorie innerhalb einer Sitzung

3. Zweck der Datenverarbeitung

Die Verarbeitung der Daten erfolgt ausschließlich zum Zweck der:

  • Herstellung von SSH-Verbindungen zu von Ihnen konfigurierten Servern
  • Speicherung Ihrer Verbindungsprofile für wiederholte Nutzung
  • Bereitstellung der Terminal- und Dateibrowser-Funktionalität

4. Speicherung und Sicherheit

Lokale Speicherung

Alle Daten werden ausschließlich lokal auf Ihrem Gerät gespeichert. Es erfolgt keine Übertragung personenbezogener Daten an Server des Entwicklers oder Dritter.

Verschlüsselung

Passwörter können optional in der macOS Keychain gespeichert werden, die eine hardwaregestützte Verschlüsselung bietet. Verbindungsprofile werden in den UserDefaults gespeichert.

SSH-Verbindungen

Alle SSH-Verbindungen sind Ende-zu-Ende verschlüsselt gemäß dem SSH-Protokoll. Die Verschlüsselungsstärke hängt von der Konfiguration des Zielservers ab.

5. Netzwerkverbindungen

Die App stellt ausschließlich Verbindungen zu den von Ihnen konfigurierten SSH-Servern her. Der Entwickler hat keinen Zugriff auf:

  • Die Inhalte Ihrer SSH-Sitzungen
  • Die übertragenen Befehle oder Dateien
  • Ihre Authentifizierungsdaten
  • Die Adressen Ihrer Server

6. Keine Analyse, Tracking oder Werbung

Diese App verwendet keine:

  • Analyse- oder Tracking-Dienste (Google Analytics, Firebase, etc.)
  • Crash-Reporting-Dienste
  • Werbenetzwerke
  • Telemetrie oder Nutzungsstatistiken

Es werden keinerlei Daten an den Entwickler oder Dritte übermittelt.

7. Ihre Rechte nach DSGVO

Sie haben folgende Rechte bezüglich Ihrer personenbezogenen Daten:

  • Auskunftsrecht (Art. 15 DSGVO): Alle Daten sind lokal auf Ihrem Gerät einsehbar
  • Recht auf Berichtigung (Art. 16 DSGVO): Sie können alle gespeicherten Daten in der App bearbeiten
  • Recht auf Löschung (Art. 17 DSGVO): Durch Löschen von Profilen oder Deinstallation der App
  • Recht auf Datenübertragbarkeit (Art. 20 DSGVO): Profile können exportiert werden

Da keine Daten an externe Server übertragen werden, haben Sie vollständige Kontrolle über Ihre Daten.

8. Datenlöschung

Alle von der App gespeicherten Daten können wie folgt gelöscht werden:

  • Einzelne Profile: Über die Profilansicht in der App löschen
  • Alle Daten: Deinstallation der App entfernt alle gespeicherten Daten
  • Keychain-Einträge: Können über die macOS Schlüsselbundverwaltung gelöscht werden
