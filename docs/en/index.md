### Features

- **Real terminal emulation**: SwiftTerm-based terminal with ANSI/VT support (e.g. `nano`, `vim`, `htop`).
- **Integrated file browser**: Remote and local file browser, open folders in the terminal, upload/download.
- **Authentication**: Password, SSH key, and passphrase.
- **Multi-tab**: Multiple parallel connections.
- **Profiles**: Save and reuse server configurations.
